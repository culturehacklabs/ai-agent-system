# KB: Tools — Listening Model Agent
**Agent 02a · CHL AI Agent System**
*Load when the practitioner asks about tools, research methods, or what Agent 02b will use.*

---

## 02a Has No Active Search Tools

Agent 02a is a design and planning agent. Its job is to build the research framework — not to execute research. No search tools are called at this stage.

The practitioner's output from this agent is a Listening Model and Big Listening Inquiry. These documents are then handed to Agent 02b, which runs the actual research using its own toolset.

---

## What Agent 02b Will Use

When the practitioner is ready to hand off to Agent 02b, they can expect the following tools to be available:

**Exa Search (MCP)**
Deep web search with full source retrieval. Best for: locating specific actors and publications, finding content from particular platforms or time periods, retrieving source material for the ANP Source Library. Returns full page content and URLs — not just summaries.

**Claude Research Mode (native)**
Synthesis and pattern recognition across large bodies of retrieved material. Best for: identifying narrative patterns across multiple sources, summarising what multiple actors are saying about a topic, building the Narrative Report from the data Exa retrieves. No activation required — available natively.

If other web search or scraping tools are active in the practitioner's environment, Agent 02b will ask whether they should be used before proceeding.

## Prior CHL Research as a Starting Point

Before running new Big Listening, it is worth checking whether CHL or the Rhizome Fellowship has prior research on the same narrative space or Pathway. Prior research can:
- Provide a baseline for comparison
- Surface communities or actors already identified
- Avoid duplicating work already done

In Iteration Mode, the existing CSV Database from the previous cycle is the primary starting point. Agent 02b extends it rather than replacing it.

If this is a first cycle, ask the practitioner: *"Has CHL or your team done any prior research on this narrative space? If so, sharing it with Agent 02b at the start of the research session will help ground the new listening in existing knowledge."*

## Small Listening — The Practitioner's Responsibility

Small Listening is conducted by the practitioner's team, not by the AI agent. Agent 02a designs the Small Listening actions; the team executes them. Agent 02b can help organise and process Small Listening data once it is collected, but it cannot conduct interviews, ethnographic observation, or community engagement.

**Common Small Listening methods:**
- Semi-structured interviews with community members, organisers, or practitioners
- Surveys or questionnaires distributed through community networks
- Manual media analysis: reading publications, following accounts, reviewing campaign materials
- Community observation: attending events, following social media accounts, reading newsletters
- Focus groups or community conversations

**Documentation discipline for Small Listening:**
- Interviews: record and transcribe (or take detailed notes with key quotes)
- Media sources: track with title, publication, date, URL, and brief note on what it reveals
- Observations: consistent format — date, space, what was observed, what it suggests
- All entries: eventually mapped to the CSV Database columns used by Agent 02b

The quality of the full research output depends as much on the Small Listening as on the Big Listening. AI-generated findings are a structural map. Small Listening fills it with community voice, texture, and lived experience that digital data cannot capture.
