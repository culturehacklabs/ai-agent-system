## How to Use This Guide

Agent 03 works best when given the complete Agent 02b package: Narrative Report, CSV Database, ANP Source Library, and completed Handoff Checklist. The more of the Handoff Checklist is answered, the faster and more accurately analysis proceeds.

---

## Starter Prompts

### Full Analysis — Complete 02b Package

Use this when you have everything from Agent 02b and are ready for full analysis.

> *"Here is my Agent 02b package: [attach/paste Narrative Report, ANP Source Library, and completed Handoff Checklist]. Run the full Understand analysis — ANP, map check-in, language analysis, Window of Discourse, and Strategic Insights / Narrative Intentions."*

---

### Full Analysis — Incomplete Handoff Checklist

Use this when some Handoff Checklist questions are unanswered but you want to proceed.

> *"Here is my Agent 02b package: [attach/paste]. The Handoff Checklist has some unanswered questions — I'll note them as gaps. Please flag where these gaps affect the analysis and proceed with what we have."*

---

### Learning Mode

Use this when you want to understand each analytical step as you go.

> *"Learning mode. Here is my Agent 02b package: [attach/paste]. Walk me through each analytical step — explain the reasoning behind ANP, community mapping, and language analysis before we proceed."*

---

### Iteration Mode

Use this when returning for a second or later cycle.

> *"Iteration mode. Here are my existing Community Profiles from Cycle 1: [attach/paste]. Here is my new Agent 02b package from Cycle 2: [attach/paste]. Update the profiles using [Updated] markers and produce a revised Community Analysis Report and Narrative Map."*

---

## Power Prompts

### Trigger the Map Check-in Explicitly

Use when you want to go straight to the map review — for example when you have an existing map and just want to update positioning.

> *"Skip to the Map Check-in. Here is the current community list with my initial positioning thoughts: [paste]. I want to use the [Lite / Visual] pathway."*

---

### Deeper ANP on a Specific Community

Use when the standard ANP feels thin for a particular community.

> *"The ANP assessment for [community name] feels thin. Can you go deeper on [Attention / Network / Power]? Are there additional Folio IDs in the ANP Source Library that should be included?"*

---

### Request More ANP Sources from Agent 02b Before Proceeding

Use when the ANP Source Library doesn't have enough for a specific community.

> *"The ANP Source Library only has [X] entries for [community name] and I don't think that's enough for a solid analysis. Before we proceed, I need to go back to Agent 02b and request more entries for this community. What should I ask for?"*

---

### Request the Visual Map Check-in Pathway

Use when you specifically want to use the CHL Narrative Mapper tool.

> *"For the Map Check-in, I want to use the Visual pathway — produce the JSON so I can import it into the CHL Narrative Mapper tool."*

---

### Request Language Analysis Only

Use when ANP is already done and you want to focus on frames and metaphors.

> *"The ANP is complete. Run Language Analysis only for these communities: [list]. Focus especially on [frames / metaphors / what is absent]."*

---

### Request Strategic Insights / Narrative Intentions Summary Only

Use when the full analysis is done and you want a focused brief for Agent 04.

> *"The Community Analysis Report is complete. Produce the Strategic Insights / Narrative Intentions section only — a focused analytical brief on what the analysis points toward for Agent 04."*

---

### Flag Insufficient Data

Use when you suspect the research base is too thin for reliable analysis.

> *"I'm concerned the data for [community name] is too thin. What would you need from Agent 02b before this community can be reliably analysed? Should I go back and request more sources?"*

---

## Prompt Tips

**Share the Handoff Checklist answers before starting.** Even partial answers make analysis significantly more accurate — especially on community identification and Big/Small Listening divergences.

**Choose your Map Check-in pathway before the session starts.** Lite (table) if you're comfortable with numbers and want speed. Visual (mapper tool) if you want to see the spatial picture and drag communities directly.

**Review Language Analysis before passing to Agent 04.** This is the interpretive layer — the agent's reading of frames and metaphors is a starting point, not a final verdict. Your community knowledge should shape this section.

**In iteration, be specific about what changed.** The more you tell the agent about what the hack revealed, the more precisely the `[Updated]` markers will reflect real change vs. stable patterns.
