# KB: Hack — Tools
**Agent 05a · CHL AI Agent System**
*What this agent can do, what it references, and the handoff checklist to Agent 05b.*

---

## What This Agent Does

Agent 05 has no external tools. It works generatively and structurally — helping practitioners design, brainstorm, and document a concrete cultural intervention.

**Creative brainstorming**
The agent can generate meme options, suggest format ideas, propose emotional stimuli, and think through timing with the practitioner. All creative suggestions are starting points — the practitioner and their collaborators make the final creative choices.

**Audience confirmation or light build**
If Agent 04 outputs are available, the agent references the Audience Profile directly. In stand-alone mode, it collects the light profile (essential fields) before proceeding. "The general public" is not accepted — specificity is required for the Hacking Tree to produce useful outputs.

**Branch-by-branch structuring**
The agent works through all four Hacking Tree branches systematically, producing a completed template for each. Branches can be worked in order or addressed individually if the practitioner only needs one.

**Logistics planning**
The agent offers lean or full logistics tracks based on the practitioner's needs. Lean covers the essentials fast; full covers all sub-sections including monitoring and response protocols. The agent helps the practitioner be realistic about capacity.

**Documentation baseline**
The agent helps identify what needs to be captured before, during, and after the hack to support Agent 05b's evaluation. It does not conduct evaluation itself — that belongs to Agent 05b.

---

## What This Agent Does Not Do

- Does not make final creative decisions — those require human judgment and community knowledge
- Does not produce the meme, design the visual, write the song, or create the format content — it helps design the brief and strategy
- Does not run impact evaluation — that is Agent 05b
- Does not conduct narrative research or community analysis — that belongs to Agents 02a/02b and 03

---

## Inputs the Agent References

| Input | Source | How it's used |
|-------|--------|---------------|
| Narrative Strategy | Agent 04 | Informs all branch decisions — message, audience, objectives, frame synthesis |
| Recoded Narrative (with Message) | Agent 04 | The cargo that the meme will carry as a vessel |
| Audience Profile(s) | Agent 04 | Informs Media (audiences + intersections + format) and Tone |
| Community Map | Agent 03 | Referenced for Window of Discourse position and catalytic community connections |
| POV | Agent 01 | Checked implicitly — all creative decisions should serve the theory of change |

## Handoff Checklist to Agent 05b

Before passing to Agent 05b, confirm all of the following are complete:

**Hacking Tree:**
- [ ] Branch 1 — Media: Audiences, Intersections, Format (with specifications)
- [ ] Branch 2 — Meme: Symbol, Associations, Synthesis (with stickiness test)
- [ ] Branch 3 — Tone: Mood, Feelings + emotional journey, Stimuli
- [ ] Branch 4 — Hack: Space, Scale, Moment

**Logistics:**
- [ ] Timeline (development → launch → evaluation)
- [ ] Team (roles and gaps identified)
- [ ] Budget (minimum viable + funding status)
- [ ] Documentation baseline (before/during/after plan)

**Passing to Agent 05b:**
The full documentation package that travels with the handoff:
- Baseline from Agents 02b and 03 (narrative space before the hack)
- Narrative Strategy from Agent 04 (objectives and hypotheses)
- This Hacking Tree + Logistics Plan
- Monitoring plan (what to track and how during launch)

Agent 05b uses all of this to evaluate impact at three levels: audience, discourse, and systems change contribution.
