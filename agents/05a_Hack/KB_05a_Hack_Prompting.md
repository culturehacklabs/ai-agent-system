## Power Prompts

**To brainstorm meme options:**
> "Give me three possible meme forms for this message: [paste message]. Audience: [description]. Evaluate each against the stickiness test."

**To sharpen the tone:**
> "My hack feels too abstract / too angry / too safe. Help me reconsider the tone. The audience is [description] and the current emotional climate is [description]."

**To pressure-test the space/scale/moment:**
> "I'm planning to launch in [space] at [scale] on [date]. Help me think through whether this is the right combination — and whether there's a better wave to catch."

**To run a stickiness test:**
> "Here's my meme: [paste]. Run it through the SUCCESs stickiness test and tell me what to strengthen."

**To choose between lean and full logistics:**
> "Help me decide whether lean or full logistics is right for this hack. Here's our situation: [describe team size, timeline, budget constraints]."

**To build a documentation baseline:**
> "Help me identify what to capture before, during, and after this hack so Agent 05b has what it needs for evaluation."

## Stuck-Point Guidance

**"I don't know what format to choose"**
Return to the Intersections step. What do both/all audience segments have in common — platforms, aesthetics, cultural references? The format decision should follow naturally from what the audience already uses and trusts. If still unclear: what format can the team actually produce? Start there.

**"My meme doesn't feel sticky"**
Run the SUCCESs test and identify which element is weakest. Common problems: too abstract (fix with Concrete), too obvious (fix with Unexpected), too logical (fix with Emotional or Story). Sometimes the symbol isn't right — try a different existing cultural object and run the Association remix again.

**"I can't find the right symbol"**
Ask: what objects, words, images, or gestures does the audience already use to talk about *adjacent* topics — nature, community, the body, home, resistance? The symbol doesn't have to come from the issue itself. Some of the most powerful memes remix symbols from completely different cultural contexts.

**"The tone feels manipulative"**
Apply the empowerment test: does this emotion lead to action, or does it leave people in feeling? Fear, guilt, and despair can mobilise — but they can also paralyse. Ask: what feeling would make this audience want to *do* something? Reorient toward determination, connection, or possibility if the current tone is closing people down.

**"The logistics feel overwhelming"**
Switch to lean track. Build the minimum viable plan: what is the simplest version of this hack that the team can actually launch? Scale ambition to real capacity. A smaller hack that ships is worth more than a larger one that doesn't.

**"I don't know when to launch"**
Run the wave assessment: is there any political moment, movement surge, anniversary, or cultural event coming up that the hack could connect to? If yes — is the team ready to catch it? If no — set a planned date and build preparation time backwards from it.

## Mode Activation

**Activate Learning Mode:**
> "Help me understand the Meme branch before we work through it — learning mode."
> "Walk me through the Tone branch step by step — learning mode."

**Activate Stand-alone Mode:**
> "I don't have Agent 04 outputs — let's build the Hacking Tree from scratch."
> "Stand-alone mode — here's my audience: [description]."

**Request a specific branch only:**
> "Just the Meme branch — I have everything else."
> "Skip to Logistics — the creative branches are done."

**Request the Hacking Tree as a creative brief:**
> "Format the complete Hacking Tree as a brief I can hand to my creative team."
